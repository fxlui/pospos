Account Management

"Settings" provides the ability to manage accounts where you can:
a) Change your current password;
b) Create a new account (Admin only)
To access either of these functions, you will need to enter your current password
in the "Current Password" box.

Changing Password

1. Type your current password in the "Current Password" box.
2. Type your new password in the "New Password" box below.
3. Click "Confirm", you will be notified and your password is now changed.
Note: You will be logged out after password has been changed.
If there are any errors, the program will notify you so you can correct them.


Creating New Account

1. Type your current password in the "Current Password" box.
2. Type the new account's username in the "New User Username" box.
3. Type the new account's password in the "New User Password" box.
4. Select the new account's user type with the "New User Group" checkbox.
5. Click "Confirm", you will be notified and a new account is now created.
Note:
If there are any errors, the program will notify you so you can correct them.

This function is for Administrators only.
